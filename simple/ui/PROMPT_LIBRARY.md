## üí° Usage Tips

### Effective Prompt Writing
- Be specific about what you want to analyze
- Include context about your industry or jurisdiction
- Ask follow-up questions for deeper analysis
- Combine multiple aspects (risk + compliance + efficiency)
- Reference specific regulations or standards
- Ask for actionable recommendations
- Request prioritized action items
- Seek implementation guidance

### File Upload Best Practices
1. Upload relevant documents (policies, procedures, data samples)
2. Use specific prompts from this library
3. Ask follow-up questions in the chat
4. Request clarification on complex responses
5. Ask for implementation steps and timelines
6. Seek examples and templates
7. Request risk assessments and mitigation strategies
8. Ask for regulatory mapping and compliance checklists
